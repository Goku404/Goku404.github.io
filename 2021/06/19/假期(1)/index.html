2、逆转链表

  1. 就地逆置一个单链表
    例子:
     h:  1 5 3 4 
 =>
 h: 4 3 5 1
    要求：
     不能新申请结点的空间

​ 算法一:
​ 1. 把原链表上的结点一一(从第一个到最后一个)摘除下来
​ 2. 把摘除下来的结点，按“头插法”加入到新链表中

​ p 指向新摘除的结点
​

​ 算法二:
​ 把原链表“一分为二”：
​ 第一个结点为链表1， h1指向第一个链表的第一个结点
​ 其余结点为链表2 h2指向后面那个链表的第一个结点

​ 只考虑h1和h2这两个结点，逆置，h2放置在h1的前面

​ p = h2->next; //先保存后面那个链表的第二个结点
​ h2->next = h1;
​ h1 = h2;
​ h2 = p;

​ h1 = h;
​ h2 = h->next ;
​ h->next = NULL;

​ while (h2)
​ {
​ p = h2->next; //先保存后面那个链表的第二个结点
​ h2->next = h1;
​ h1 = h2;
​ h2 = p;

​ }

双链表的打印方式可以用递归的方法实现 先递归打印 p->next，然后打印本身
