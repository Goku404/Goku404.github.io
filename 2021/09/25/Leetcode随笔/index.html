一.数组

1.查找缺失数字

题目描述 : 给定一个长度为 n 的数组，其中包含范围为 1 到 n 的整数，有些整数重复了多次，有些整数 没有出现，求 1 到 n 中没有出现过的整数。

Input: [4,3,2,7,8,2,3,1] Output: [5,6]

解析：遍历两次数组，第一次把原数组的相对位置变为负数，第二次遍历把正数位置编号压入数组，遍历得出结果

2.旋转矩阵

题目描述 给定一个 n × n 的矩阵，求它顺时针旋转 90 度的结果，且必须在原矩阵上修改（in-place）。 怎样能够尽量不创建额外储存空间呢？

3.二维排序数组查找值

这道题有一个简单的技巧：我们可以从右上角开始查找，若当前值大于待搜索值，我们向左 移动一位；若当前值小于待搜索值，我们向下移动一位。如果最终移动到左下角时仍不等于待搜 索值，则说明待搜索值不存在于矩阵中。

2.单调栈

单调栈通过维持栈内值的单调递增(减)性，在整体O(n)的时间内处理需要大小比较的问题

题目描述:给定每天的温度，求对于每一天需要等几天才可以等到更暖和的一天。如果该天之后不存在 更暖和的天气，则记为 0。

思路:构造一个单调栈，压入温度数组的序号，遍历温度数组，如果温度比栈顶元素的下标对应温度大的话，出栈，直到栈空或者比栈顶小，并序号入栈，vector 初始化默认值为 0
